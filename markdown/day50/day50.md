### 1. react

- react : 프론트엔드 프레임워크
- path : `node --version`
- cmd : 명령프롬프트, cd 작업폴더
- react : `npx create-react-app my-app1`
- 목록보기 : `dir`
- 해당 폴더로 이동 : `cd my-app1`롯한 모던 자바스크립트 웹 시스템이 작성되는 방식, html 파일은 하나만 사용
- 리액트 실행하기 : `npm start`
- SPA(single page application) : 리액트를 비용하고 자바스크립트를 이용해 화면을 전환함으로서 화면이동 등의 동작을 표현
- 리액트 구조
  - `index.js` : 시작점
