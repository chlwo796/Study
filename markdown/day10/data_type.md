## Data Type

- __primitive type(기본 타입)__
  - 정수타입
    - byte
    - char
    - short
    - int
    - long
  - 실수타입
    - float
    - double
  - 논리타입
    - boolean

- __reference type(참조 타입)__
  - 배열타입
  - 열거타입
  - 클래스
  - 인터페이스

- __변수의 메모리 사용__
  - 기본 타입 변수 : 실제 값을 변수 안에 저장
  - 참조 타입 변수 : 주소를 통해 객체 참조
